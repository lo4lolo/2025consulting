자가진단 분석 자동화 웹앱

교원 자가진단 결과를 분석하고, 데이터 기반의 맞춤형 연수 계획 수립을 지원하여 컨설팅의 효율성과 전문성을 극대화하는 솔루션입니다.

무엇을 자동화 할까요?

데이터 분석 및 시각화

수십 개의 응답 데이터를 클릭 한 번으로 분석합니다. 영역별 평균 점수, 수준(도약/만족/추월)을 계산하고 레이더 차트로 시각화하여 한눈에 현황을 파악할 수 있습니다.

맞춤형 피드백 생성

미리 구축된 피드백 DB를 기반으로, 분석된 영역별 수준에 맞는 진단 코멘트와 추천 연계 프로그램을 자동으로 생성합니다. 모든 학교는 자신만의 맞춤 보고서를 받게 됩니다.

최적 연수 패키지 추천

학교의 전체 역량 분포(기초, 수업혁신, 공동체)를 종합적으로 분석하여, 7가지 유형의 모듈 패키지 중 가장 적합한 패키지를 이미지와 함께 추천합니다.

어떻게 작동하나요?

1. 교원 수 입력

전체 교원 수를 입력하고 '분석 실행' 버튼을 클릭합니다.

→
2. 자동 분석 (Apps Script)

구글 시트의 응답 데이터를 Apps Script가 실시간으로 분석합니다.

→
3. 결과 보고서 확인

시각화된 차트, 상세 피드백, 추천 패키지가 포함된 보고서를 즉시 확인하고 연수 계획을 수립합니다.

기대 효과

컨설팅 시간 단축

데이터 취합 및 분석에 소요되던 시간을 획기적으로 줄여, 컨설턴트가 심층 면담과 질 높은 피드백 제공에 더 집중할 수 있도록 돕습니다.

데이터 기반 컨설팅

컨설턴트의 직관이나 경험을 넘어, 객관적인 데이터 분석 결과를 바탕으로 학교의 강점과 약점을 진단하고 맞춤형 성장 계획을 제안할 수 있습니다.

보고서 품질 표준화

모든 학교에 일관되고 전문적인 형식의 보고서를 제공하여 컨설팅의 신뢰도를 높이고, 학교 현장의 만족도를 향상시킬 수 있습니다.
